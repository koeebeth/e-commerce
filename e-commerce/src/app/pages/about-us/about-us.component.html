About Site Developers

This project is the result of our collective efforts, with each member contributing equally.
The project was developed using: HTML, SCSS, TypeScript, Angular for frontend part
and CommerceTools for Backend part.
So in this project we organized the work in the way that each team member could accomplish various tasks , thereby we learned more different things: whether it was integration with commercetools via API requests, creating new components, work with routing or designing and implementing design solutions and so on.
For synchronization and control over the development stages, we used a Jira board, as well as regular meetings and daily updates in Discord.
